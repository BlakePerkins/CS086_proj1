Command *col* Syntax *col* Example Description *col*
balance	*col* balance *col* shows the balance *col*
history	*col* history *col* prints list of transactions *col*
inventory *col* inventory *col* prints available items with name and ID *col*
add item <str> <int> <float> *col* add item chips 2 $1.00 *col* add an item name qty price *col*
buy item <str> {5}<int>	*col* buy item chips 1 2 2 4 3 *col* buys an item with # dollars, quarters, dimes, nickles,
pennies. It also shows change given and the remaining
balance with currency distribution. For change, the machine
uses the largest denominator of currency that is available. *col*
help *col* help *col* display help menu with these commands *col*
exit *col* exit *col* exit the vending machine
